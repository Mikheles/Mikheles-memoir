# זיכרונות סבא ותולדות המשפחה

אתר משפחתי המאגד את הזיכרונות בשלוש תצוגות (עברית / מקור ברוסית / צד-בצד),
צילומי מסמכים מקוריים, עץ משפחה וציר זמן.

## תצוגת הקריאה
בעמוד "הזיכרונות" יש מתג תצוגה למעלה:
- **עברית** — התרגום בלבד
- **מקור ברוסית** — צילומי כתב היד המקורי (247 עמודים)
- **צד בצד** — התרגום העברי לצד עמוד כתב היד התואם

לחיצה על עמוד כתב יד מגדילה אותו.

## מבנה הקבצים
- `index.html` — הדף הראשי
- `styles.css` — העיצוב
- `app.js` — הלוגיקה (קריאה רב-לשונית, גלריה, עץ, ציר זמן)
- `data.js` — טקסט הזיכרונות + מיפוי עמודי כתב היד (נוצר אוטומטית)
- `tree-data.js` — **עץ המשפחה וציר הזמן — כאן מוסיפים/מתקנים שמות ותאריכים**
- `assets/docs/` — צילומי המסמכים (נספחים)
- `assets/manuscript/` — 247 עמודי כתב היד המקורי (p-001.jpg … p-247.jpg)

## איך מעלים ל-GitHub Pages (חינמי)
1. צרו חשבון ב-github.com.
2. צרו repository חדש, למשל `family-memoir`.
3. העלו את כל הקבצים מהתיקייה הזו (commit + push). שימו לב: תיקיית
   `assets/manuscript` כוללת 247 תמונות (~30MB) — זה בסדר ל-GitHub.
4. Settings ← Pages ← Source: branch `main`, תיקייה `/ (root)` ← Save.
5. אחרי דקה-שתיים האתר יהיה בכתובת:
   `https://<שם-המשתמש>.github.io/family-memoir/`

## עריכת עץ המשפחה
פתחו את `tree-data.js` בעורך טקסט. כל אדם הוא אובייקט עם `name`, `years`, `note`, `desc`.

## התאמת עמודי כתב היד
המיפוי בין פרק לעמודי כתב היד מחושב לפי מספרי העמודים שבטקסט והוא
שמור בתוך `data.js` בשדה `ms_pages` של כל פרק. אפשר לערוך ידנית אם צריך לדייק.
